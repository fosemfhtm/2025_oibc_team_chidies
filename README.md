# 2025 OIBC Team CHIDIES

한국어로 작성된 태양광 발전량 추정 대회 코드 모음입니다.

폴더 구조:

- `final_codes_TEAM_CHIDIES/` : 발표 및 제출용 모델 코드
- `태양광 발전량 추정 전략_TEAM CHIDIES.pdf` : 프로젝트 설명/발표자료

간단 사용법:

1. 아래 의존성 설치:

```bash
pip install -r requirements.txt
```

2. `final_codes_TEAM_CHIDIES` 폴더로 이동 후, `info.md` 파일의 실행 순서를 따라 전처리 및 모델 스크립트를 실행하세요.

3. 예시(베이스라인 실행):

```bash
python final_codes_TEAM_CHIDIES/0_baseline.py
```

데이터

`train.csv`, `test.csv`, `submission_sample.csv` 파일을 프로젝트 루트(또는 `final_codes_TEAM_CHIDIES/` 내부)로 추가한 뒤 스크립트를 실행하세요.
